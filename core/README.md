# Readium (_core_)

### Useful commands:
##### Install dependencies
```
npm install
```

##### Runs all tests
```
npm run test
```

##### Lints and fixes files
```
npm run lint
```

##### Compiles for production
```
npm run build
```

##### Runs server (without build)
```
npm run dev
```

##### Runs server (with build)
```
npm run start
```
